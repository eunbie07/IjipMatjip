## 1. 전체 스프린트 목표

> 사용자가 업로드한 방 사진을 기반으로 YOLOv11, MiDaS, RoomNet 등의 AI 모델을 활용해 가구를 감지하고, 깊이·레이아웃 정보를 추정하여 실제 방의 평면도(2D) 구조를 자동으로 시각화하는 웹 서비스를 개발한다. 
> 주요 작업에는 모델 연동, 프론트 UI/UX 구성, API 구축, 도커/쿠버네티스 배포 및 CI/CD 자동화가 포함된다.
